Sobre mí

Soy Álvaro Zambianchi, desarrollador web centrado en HTML, CSS y JavaScript. Trabajo con esfuerzo y dedicación para que cada página que toco funcione como debe.

Mi especialidad es encontrar y eliminar errores. Ese bug que lleva semanas sin resolverse, ese layout que se rompe en móvil, ese script que no carga: eso es lo que hago.

Actualmente sigo formándome cada día — porque en desarrollo web siempre hay algo nuevo que aprender y eso es exactamente lo que me apasiona.

"Cada línea de código corregida es un problema menos para ti y un paso más para mí."
